## 👋 About Me

 Working as a Data Engineer at **Flipkart (Cleartrip)** on large-scale GCP pipelines. Previously a **Research Engineer at Siemens**, where I built **DANA** — a neuro-symbolic LLM-powered agent for industrial anomaly detection.

My work sits at the intersection of **Computer Vision**, **Federated Learning**, **Agentic AI**, and **MLOps**. I've published at ICPR 2024 and ICIT 2025, with a paper currently under review at ACMM 2026.

- 🔬 &nbsp;Thesis: Privacy-preserving federated face recognition (FedFace) with FedMoCo self-supervised learning
- 🤖 &nbsp;Built DANA at Siemens — a BDI/OODAR neuro-symbolic agent using LangGraph + Gemini for chiller anomaly detection
- ☁️ &nbsp;Building GCP-scale data pipelines at Flipkart/Cleartrip
- 🚀 &nbsp;Founding two startups: **Chef.ai** (AI-powered Indian cooking with YOLO-Seg + Gemini) and a **WhatsApp-based rural healthcare diagnostic tool** for ASHA workers
- ♟️ &nbsp;International chess player & robotics enthusiast

## 🚀 Featured Projects

### 🍳 Chef.ai
AI-powered Indian cooking platform using YOLO-Seg + Gemini for ingredient detection and a six-agent LangGraph recipe pipeline. Handles ingredient segmentation, nutrition analysis, and personalized recipe generation end-to-end.

### 🏥 Rural Healthcare Diagnostic Tool *(Stealth)*
WhatsApp-based multimodal diagnostic assistant for ASHA workers in rural India. Built on federated learning to preserve patient privacy, with on-device inference to handle low-connectivity environments.

### 🤖 DANA (Siemens)
Domain-Aware Neuro-symbolic Agent for industrial chiller anomaly detection. Built with LangGraph, Gemini, and a BDI/OODAR architecture — combining symbolic reasoning with LLM intelligence for explainable fault diagnosis.

### 🔐 FedFace
Privacy-preserving federated face recognition with multi-task learning and FedMoCo self-supervised learning. Thesis research presented at ICIT 2025.
